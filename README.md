# Defense of the Craft
Dota-based Minecraft mod which brings artifacts and unique mechanics from the best-known MOBA game developed by Valve Software!

**THIS MOD IS NOT AFFILIATED WITH VALVE SOFTWARE AND ALL DOTA AND STEAM ASSETS ARE PROPERTY OF VALVE SOFTWARE!!!**

## Warning
I'm not a Minecraft mod developer, artist or a game designer. I develop this mod according to my personal taste and to what I think is fun and entertaining.

I don't plan to release this mod for different versions of the game because I don't have either will or time for it. The project is open-source, so you're welcome to fork it or make changes via pull requests.
